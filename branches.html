GitHub Personal Access Token

현재 저장소 의 브랜치 / 커밋 / PR 데이터를 받기 위한 토큰입니다.
토큰은 이 브라우저 localStorage 에만 분리 저장 (저장소별 다른 key) 되며 외부로 전송되지 않습니다.

발급 방법: GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token → Resource owner 본인 → Repository access = 해당 저장소 → Repository permissions: Contents: Read-only, Metadata: Read-only, Pull requests: Read-only.
