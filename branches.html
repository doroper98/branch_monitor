GitHub Personal Access Token

현재 저장소 를 포함해, 모니터링 대상 저장소들의 브랜치 / 커밋 / PR 데이터를 받기 위한 토큰입니다.
토큰은 이 브라우저 localStorage 에만 저장 되고 외부로 전송되지 않습니다. 한 디바이스 내에서는 모든 저장소가 같은 키 (branch_monitor.token.v1) 를 공유하므로 한 번만 입력하면 됩니다. 다른 디바이스에서는 다시 입력해야 합니다 (localStorage 는 디바이스 / 브라우저 단위).

발급 방법: GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token → Resource owner 본인 → Repository access = 모니터링할 저장소 모두 선택 → Repository permissions: Contents: Read-only, Metadata: Read-only, Pull requests: Read-only.
